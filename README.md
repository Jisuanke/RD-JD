- 1 年及以上后端开发相关经验
- 熟悉一种 Web 后端开发语言，或了解至少一种主流 Web 开发框架
- 有一定的 Python/Bash 脚本的编写能力，熟悉使用 Git 和工程管理工具
- 熟悉 PHP / Laravel 框架者优先
- 扎实的数据结构知识和算法分析能力，有 OI / ACM-ICPC 类算法比赛经验者适度放宽上述要求
